Materials Showing Pink Color?
If you imported this into a URP project or a Universal 3D template you may see all objects in pink. To fix this, simply go to the SRP folder and double click on URP_Materials and import. Or you could select all materials from the materials folder then go to Edit menu -> Rendering -> Materials -> Convert Selected Built-in Materials to URP.
